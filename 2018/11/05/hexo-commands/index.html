一些常会用到的指令

初始化网站

$ hexo init [folder]
folder用以指定初始化目录的位置，若无指定则默认当前位置

新建文章

$ new [layout] <br>layout指定文章类型post、page、draft，若无指定则默认由_config.yml文件的default_layout决定<br>指定 draft 参数，文件将保存到 source/_drafts 文件夹中<br>将草稿移动到 source/_posts文件中:$ hexo publish [layout] <filename><br>title指定文章标题，如果title中含有空格需要使用双引号包围 </filename>

生成静态文件

$ hexo generate
$ hexo g

启动本地服务器

$ hexo sever
$ hexo s
以 debug 模式开启 Hexo 本地服务器预览：$ hexo server –debug
debug 模式下可以查看实时改动情况

部署网站

$ hexo deploy
生成后直接部署： $ hexo generate –deploy

清理文件

$ hexo clean
清除缓存文件db.json和生成文件public，网站显示异常时可尝试此操作

使用淘宝镜像

npm太慢时可以使用淘宝镜像

临时使用

npm –registry https://registry.npm.taobao.org install express

永久使用

npm config set registry https://registry.npm.taobao.org
可以通过npm config get registry或npm info express来验证是否成功

通过cnpm使用

npm install -g cnpm –registry=https://registry.npm.taobao.org
通过cnpm install XXX 来使用
